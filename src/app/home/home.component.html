Gás do Almir
Quem somos

A página gás do almir é uma idéia que surgiu da necessidade de realizar o controle de vendas de gás do nosso negócio aliado ao trabalho de conclusão do curso de Análise e desenvolvimento de sistemas da ULBRA - Universidade Luterana do Brasil, câmpus Canoas/RS.

As tecnologias utilizadas para desenvolver essa aplicação foram:
- Frontend: Angular 10 / Html / Css / TypeScript
- Backend: Java 8 / Springboot 2.1.8 / JsonWebToken
- Banco de dados: PostgreeSql

Agenda

O objetivo desta página é realizar o controle e o armazenamento on line dos clientes da empresa e das vendas realizadas, facilitando o controle financeiro e a visualização das operações realizadas. O projeto tem por objetivo ser simples e intuitivo facilitando a utilização do usuário final, pois o mesmo não não está familiarizado com esse tipo de tecnologia.
